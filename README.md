# She Shield AI 🚨

She Shield AI is an AI-powered women safety web application designed to provide real-time emergency support using AWS cloud services.

## 🔥 Features
- One-click SOS alert
- Live location capture
- Voice command support
- AI Safety Assistant
- Multilingual chatbot (Tanglish support)

## ☁️ AWS Services Used
- Amazon S3 (Website Hosting)
- AWS Lambda (Backend Processing)
- Amazon API Gateway (API Integration)

## ⚡ How It Works
User → S3 Website → API Gateway → Lambda → Response

## 🚀 Future Scope
- Amazon SNS (SMS alerts)
- Amazon DynamoDB (Data storage)
- Amazon Transcribe (Speech processing)
- Amazon Lex (AI chatbot)

## 🛠️ Built Using
- HTML, CSS, JavaScript
